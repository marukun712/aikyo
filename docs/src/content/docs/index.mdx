## aikyoとは

aikyoは、相互につながるAIコンパニオンを作成するためのフレームワークです。

<CardGrid stagger>
  <Card title="マルチエージェント">コンパニオン同士がP2Pでつながり、リアルタイムで情報を共有します。さらに、自動ターンテイキングとクロージングにより、人間らしい自然な会話を実現します。</Card>
  <Card title="Companion Card">
    JSONベースの設計書でコンパニオンの性格、行動、判断基準を定義できます。
  </Card>
  <Card title="アクション＆ナレッジ">外部APIとの連携や動的な知識取得が簡単に実装できます。</Card>
  <Card title="CEL式による制御">
    条件分岐をCEL式で記述し、柔軟なコンパニオンの行動を実現できます。
  </Card>
</CardGrid>
