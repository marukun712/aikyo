## aikyoとは

aikyoは、バーチャルな体を持ったAIコンパニオンを作成しやすくするためのフレームワークです。

<CardGrid stagger>
  <Card title="P2P通信">
    コンパニオン同士がP2Pでつながり、リアルタイムで情報を共有します。
  </Card>
  <Card title="Companion Card">
    JSONベースの設計書でコンパニオンの性格、行動、判断基準を定義できます。
  </Card>
  <Card title="アクション＆ナレッジ">
    外部APIとの連携や動的な知識取得が簡単に実装できます。
  </Card>
  <Card title="CEL式による制御">
    条件分岐をCEL式で記述し、柔軟なコンパニオンの行動を実現できます。
  </Card>
</CardGrid>
