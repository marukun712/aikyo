## What is aikyo?

aikyo is a framework designed for creating interconnected AI companions.

Discord Server:
https://discord.gg/JxAVZkQdDX


<CardGrid stagger>
  <Card title="A2A for Conversations">Enables companion-to-companion P2P connections with real-time information sharing. Additionally, it implements automated turn-taking and conversation closure mechanisms to enable seamless dialogue between AI companions.</Card>
  <Card title="Companion Card">
    A JSON-based specification document that allows you to declaratively define the personality, behaviors, and decision-making criteria of your companion.
  </Card>
  <Card title="Tool System">Easily integrates with external APIs and enables dynamic knowledge acquisition.</Card>
  <Card title="Control via CEL Expressions">
    Allows for flexible companion behavior through conditional logic expressed using Common Expression Language (CEL) expressions.
  </Card>
</CardGrid>
